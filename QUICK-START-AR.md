## جرب الميزات الجديدة 🚀

### تجربة 1: رقم عشوائي 🎲
1. من **"Python Libraries"** → اسحب **random.randint**
2. حدد من `1` إلى `100`
3. اربطها بـ print
4. شغّل البرنامج
5. كل مرة تشغله ستحصل على رقم مختلف!

```python
import random
print(random.randint(1, 100))
```

### تجربة 2: حساب الجذر التربيعي ✏️
1. من **"Python Libraries"** → اسحب **math.sqrt**
2. ضع الرقم `144` داخلها
3. اربطها بـ print
4. شغّل → النتيجة: `12.0`

```python
import math
print(math.sqrt(144))
```

### تجربة 3: اعرض التاريخ اليوم 📅
1. من **"Python Libraries"** → اسحب **datetime.now()**
2. اسحب **"استخرج السنة"** واربط datetime.now بها
3. اربط الناتج بـ print
4. شغّل → يظهر السنة الحالية: `2026`

```python
from datetime import datetime
print(datetime.now().year)
```
