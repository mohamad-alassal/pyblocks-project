### 2. إضافة مكتبات Python الأساسية 📚

تم إضافة كتل جديدة لأهم مكتبات Python:

#### **مكتبة Random (عشوائي)** 🎲
- `random.randint(min, max)` - رقم عشوائي بين قيمتين
- `random.choice(list)` - اختيار عنصر عشوائي من قائمة
- `random.random()` - رقم عشري بين 0 و 1

#### **مكتبة Time (الوقت)** ⏰
- `time.sleep(seconds)` - توقف البرنامج لفترة محددة
- `time.time()` - الوقت الحالي بالثواني

#### **مكتبة Math المتقدمة** 🔢
- `math.sqrt(x)` - الجذر التربيعي
- `math.pow(base, exp)` - الأس
- `math.floor(x)` - تقريب للأسفل
- `math.ceil(x)` - تقريب للأعلى
- `math.pi` - ثابت باي (π)

#### **مكتبة Datetime (التاريخ)** 📅
- `datetime.now()` - التاريخ والوقت الحالي
- استخراج السنة/الشهر/اليوم من تاريخ

#### **دوال Python المدمجة** 🛠️
- `input(prompt)` - قراءة مدخل من المستخدم
- `int(value)` - تحويل إلى عدد صحيح
- `float(value)` - تحويل إلى عدد عشري
- `str(value)` - تحويل إلى نص
- `len(value)` - طول النص أو القائمة
- `range(start, stop)` - إنشاء مدى رقمي

**الملفات الجديدة:**
- `javascript/python-blocks.js` - تعريف جميع الكتل الجديدة

**الملفات المعدلة:**
- `javascript/editor.js` - إضافة الكتل إلى صندوق الأدوات
- `editor.html` - تضمين ملف الكتل الجديدة

## كيفية الاستخدام 🚀

### تشغيل المشروع
```bash
npm install
node server.js
```

ثم افتح المتصفح على: `http://localhost:3000`

### استخدام المكتبات الجديدة
1. افتح المحرر
2. ابحث عن القسم **"Python Libraries (مكتبات بايثون)"** في صندوق الأدوات
3. اسحب الكتل التي تريدها إلى مساحة العمل
4. المكتبات ستُستورد تلقائياً في كود Python المُنتج

### مثال على الاستخدام

**مثال 1: لعبة تخمين الرقم**
```python
import random

secret = random.randint(1, 10)
guess = int(input('خمن الرقم من 1 إلى 10: '))

if guess == secret:
    print('أحسنت! خمنت الرقم الصحيح')
else:
    print('خطأ! الرقم كان: ' + str(secret))
```

**مثال 2: حساب الجذر التربيعي**
```python
import math

number = 16
result = math.sqrt(number)
print('الجذر التربيعي لـ ' + str(number) + ' هو ' + str(result))
```

**مثال 3: عرض التاريخ الحالي**
```python
from datetime import datetime

now = datetime.now()
print('السنة: ' + str(now.year))
print('الشهر: ' + str(now.month))
print('اليوم: ' + str(now.day))
```
